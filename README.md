# ---
本项目旨在于分享用于全球人口可视化的数据，使用的方法以及技术，巩固在科学可视化中学习到的知识技能。
## 项目数据链接来源
WPP2024宏观总量核心数据源： https://population.un.org/wpp/downloads    

GPWv4空间精细支撑数据源： https://www.earthdata.nasa.gov/data/catalog/sedac-ciesin-sedac-gpwv4-popcount-4.0  


## world.xlsx 为WPP2024选取部分并处理后的数据
## zip文件为处理数据所需要的代码
